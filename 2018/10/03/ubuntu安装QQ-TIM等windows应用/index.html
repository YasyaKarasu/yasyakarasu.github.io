ubuntu安装QQ,TIM等windows应用

嗯，Ubuntu虽然好，但是很多仅仅想着盈利而不为Linux用户着想的猥琐的公司却不推出他们的Linux版的应用，因此我们只能采用wine来模拟windows环境

安装方法：

简单安装（过程和内容不可控，但操作简单，傻瓜式）

使用 flatpak install 文件名 进行安装
先安装运行时：flatpak install com.deepin.wine.Platform.2.18-10.flatpak
再安装你需要的应用，如TIM：flatpak install com.tencent.tim.2.0.flatpak

自行构建安装（过程和内容可控，但耗时间，需要有一定的flatpak知识）

构建需要的运行时

org.freedesktop.Platform/i386/1.6
org.freedesktop.Sdk/i386/1.6

构建顺序

必须先构建并安装运行时com.deepin.wine.Platform，再构建和安装其他应用

运行

由于第一次运行需要解压wine虚拟环境到 ~/.var/app/应用ID/wine/ ，这个过程可能失败，所以建议第一次运行时使用
flatpak run 应用ID
来运行应用，这样出问题的话可以知道是什么问题。
如果第一次没问题，之后就可以在应用菜单里启动。
使用 flatpak run 应用ID -e 可以清除wine虚拟环境
使用 flatpak run 应用ID -r 可以重新解压wine虚拟环境

卸载

首先使用 flatpak run 应用ID -e 删除wine虚拟环境，这会删除你下载到虚拟环境的所有文件，然后使用flatpak uninstall 应用ID 卸载应用。
当所有你安装的deepin-wine应用卸载完后，你可以使用 flatpak uninstall com.deepin.wine.Platform卸载运行时。
