ubuntu18.04配置攻略

每次装ubuntu都要重新到百度上找一堆东西……麻烦死了……既然这次弄了个新的好看的博客，那就让它发挥一点作用吧！

软件源更改

打开应用商店，左上角点击ubuntu软件，出现的菜单中选择“软件和更新”，然后更改一下“下载自”右边的软件源即可！

推荐（假的）：

http://mirrors.tuna.tsinghua.edu.cn/ubuntu
http://mirrors.cn99.com/ubuntu

安装Chrome

打开终端，然后sudo su一下好了，方便一点。如果你这里不sudo su的话，那么接下来的4条命令都要加sudo。

输入以下命令：

wget http://www.linuxidc.com/files/repo/google-chrome.list -P /etc/apt/sources.list.d/
wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
apt-get update
apt-get install google-chrome-stable

然后在左下角的显示应用程序里面打开全部，就可以看到Chrome已经躺在里面了！

1.Numix

Numix是ubuntu下一款很好看的扁平化主题

好像我曾经百度Numix安装的时候经常看到这么一句话。当然，Numix也确实是好看。（我还是从rxz巨魔那里看到的这个主题）
（顺便给OIER们安利rxz巨魔的blog！（虽然各位OIER应该已经有了）rxz巨魔的博客，当然rxz巨爷的一个很大的贡献就是BZOJ离线题库！大家可以戳这里来访问，左侧的“关于”里面有下载离线BZOJ的途径）

安装方法如下！

打开终端！输入如下命令！（还是可以先 sudo su，这样后面就不在需要输入sudo了）

sudo add-apt-repository ppa:numix/ppa
sudo apt-get update
sudo apt-get install numix-gtk-theme numix-icon-theme-circle

然后打开gnome-tweak-tools，里面可以更换图标，和应用程序的主题。看自己喜好选择喽~
